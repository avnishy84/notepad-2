Frequently Asked Questions

What is One Notepad?

+

One Notepad is a simple, fast, and secure note-taking app with rich text editing, cloud sync, and even a built-in Snake game.

Is it free?

+

Yes, completely free. Create an account and start taking notes right away with all features included.

How are notes saved?

+

Notes auto-save to Firestore when logged in. You can also download any note as an HTML file.

Can I use Google Sign-In?

+

Yes! Click "Continue with Google" in the login modal for one-click secure access.

Is my data secure?

+

Yes. Firebase Authentication and Firestore ensure your notes are only accessible to you when logged in.
